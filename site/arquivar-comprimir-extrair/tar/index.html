tape archive

Apesar do nome "tar" ser derivado de "tape archive", não limita a fitas magnéticas. Ele se tornou largamente usado para arquivar vários ficheiros em um único, preservando informações como datas e permissões.

Apesar de ser mais comum em sistemas Unix, este formato é suportado pela maioria dos descompactadores para Windows, como por exemplo o 7-zip.

Os argumento finais do comando tar são os nomes dos ficheiros ou pastas nos quais eles podem ser arquivados. O uso de um nome nas pastas, implica sempre que as sub-pastas sob ela, sejam incluídas no arquivo.

TAR também consegue gravar a propriedade e as permissões dos arquivos. Ainda, consegue manter a estrutura de diretórios original (se houve compactação com diretórios), assim como as ligações diretas e simbólicas.

parametros descricao
-c cria um novo arquivo tar;
-M cria, lista ou extrai um arquivo multivolume;
-p mantém as permissões originais do(s) arquivo(s);
-r acrescenta arquivos a um arquivo tar;
-t exibe o conteúdo de um arquivo tar;
-v exibe detalhes da operação;
-w pede confirmação antes de cada ação;
-x extrai arquivos de um arquivo tar;
-z comprime ou extrai arquivos tar resultante com o gzip;
-j comprime ou extrai arquivos tar resultante com o bz2;
-f especifica o arquivo tar a ser usado;
-C especifica o diretório dos arquivos a serem armazenados

TAR na pratica

compactar todos os ficheiros .log

tar -cvzf todos_logs.tar.gz /var/log/*.log


O tar nao elemina os ficheiros originais!

comando para abrir ficheiro tar.gz

tar -xvzf todos_logs.tar.gz

boa pratica testar o arquivamento depois de criar um ficheiro tar # tar -tf crybit.tar

remover apenas um ficheiro num tar tar --delete -f crybit.tar file4.txt

utilizar o wildcards no nome para eleminar sequencias # tar --wildcards --delete -f crybit.tar 'file*'

gzip

o programa mais comum para comprimir ficheiros no Linux é o gzip escrito por Jeanloup Gailly e Mark Adler. Utilizado para reduzir o tamanho de um unico ficheiro. GNU Zip é publicado em 1992 para evitar problemas com outro programa utilizado mas patenteado do Unix que usa o algoritmo Lempel-Ziv-Welch (LZW).

A compressao do gzip é baseado em um precursor não patenteado de LZW chamado LZ77. O LZ77 funciona melhor. Não confundir com o PKZIP e produtos Windows simalhates.

O que o GZIP não consegue fazer, o TAR (Tape ARchives) faz. Ele é um programa capaz de armazenar vários arquivos em um só. Porém, não é capaz de compactar os arquivos armazenados. Como é possível notar, o TAR serve de complemento para o GZIP e vice-versa.

Por isso, foi criado um parâmetro no TAR para que ambos os programas possam trabalhar juntos. Assim, o TAR "junta" os arquivos em um só. Este arquivo, por sua vez, é então compactado pela GZIP. Quando ocorre o trabalho conjunto entre TAR e GZIP, o arquivo formado tem a extensão tar.gz.

  • gzip pode descomprir ficheiros .Z por causa da pantente do Unisys que so cobre o procedimento da compressao.
  • nao confudir com PKZIP ou ZIP estes ficheiros tem as caracteristicas de serem comprimidos e depois arquivados
  • gzip apenas trata da compressao e deixa a cargo o arquivamento para tar ou cpio.

  • gzip é capaz de descompactar ficheiros ZIP desde que sejam 1 ficheiro utilizando o metodo DEFLATE

  • Processa e substitui ficheiros individuais, anexando o sufixo *.gz a extenção. Esta substituição ocorre independentemente no ficheiro resultando um ficheiro menor do que o original.

Comando gzip

parametros descricao
-c extrai um ficheiro para a saída padrão
-d descompacta um ficheiro comprimido. Alternativa o gunzip
-l lista o conteúdo de um ficheiro compactado;
-v exibe detalhes sobre o procedimento;
-r compactar recursivamente pastas e seus ficheiros;
-t testa a integridade de um ficheiro compactado.
-S alterar sufixo .gz
-1 .. -9 nivel de compressao -1 rapido/má compressao, -9 lento/boa compressao

exemplos com gzip

Ao compactar um ficheiro para .gz o original é eleminado para dar lugar a outro compactado. gzip ficheiro.txt

para descompactar o ficheiro

$ gzip -d ficheiro.txt.gz

ou

gunzip ficheiro.txt.gz

Alterar o tipo de extencao .gz para outra e ver resultados em modo verbose.

gzip -vS .batatas ficheiro.txt

para descompactar um ficheiro com outro sufixo .batatas

gzip -dS .batatas ficheiro.txt

bzip2

bzip era o antecessor do bzip2 que usava outra codificacao.

Bzip2 comprime a maioria dos formatos de ficheiros de maneira mais eficiente do que o tradicional gzip ou Zip, porém de forma mais lenta. Desta maneira é claramente similar aos algoritmos de compressão de gerações recentes.

O programa não tem recursos para trabalhar com múltiplos arquivos, criptografia ou divisão de arquivos.

argumento descricao
k mantém o arquivo original (backup)
q compacta mesmo com erro
d descompacta o arquivo do tipo .bz2
f sobrescreve o arquivo já existente

compactar com tar

tar -cvjf arquivo.tar.bz2 pasta

descompactar ficheiro tipo tar.bz2

tar -xvjf arquivo.tar.bz2

zip

O compactador zip além de um utilitário para a compactação serve também como empacotador, tendo como principal função a sua portabilidade, ou seja, esta ferramenta é compatível com outros sistemas de compactação do tipo pkzip e similares.

A compactação de vários ficheiros em um só ficheiro, o zip é muito útil, pois em sua própria sintaxe é solicitado um nome para ser o ficheiro principal (não pode ser um arquivo existente) evitando assim a perda do arquivo originário.

sintaxe zip [-opção] arquivo.zip arquivo1 arquivo2 ... arquivoN

argumento descricao
r trabalha de forma recursiva sobre diretórios;
T testa a integridade do arquivo de zip;
D não adiciona diretórios quando solicitado uma pasta inteira.

zip na pratica

compactar varios ficheiros zip arquivo.zip teste teste1

descompactar ficheiro unzip arquivo.zip
