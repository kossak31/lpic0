Navegar pelo sistema e listar ficheiros

comandos descricao
pwd mostra o caminho por absoluto da pasta em que nos encontramos de momento, o pathname
cd navegar entre as pastas
ls listar ficheiros dentro das pastas

comando pwd

a prompt aponta neste momento para pasta HOME represetanda pelo caracter ~.

[username@hostname ~]$

para descobrir o caminho absoluto escreva o comando pwd.

[username@hostname ~]$ pwd
/home/username

comando ls

Para listar o conteudo dentro de uma pasta necessita de ter pelo menos a permissão de leitura. Usa o comando ls.

opcao descricao
-l usa o formato lista longa (com mais detalhes)
-a mostra os ficheiros ocultos
-A não mostra os ficheiros implicitos . e ..
-r inverte a ordem
-R listar as pastas recursivamente
-d listar atributos de pasta 
[username@hostname ~]$ ls # vista simples e lista horizotalmente
Desktop  Documents  Downloads  Music  Pictures  Public  Templates  Videos

por vezes existem ficheiros ocultos. o nome do ficheiro é iniciado com o caracter . para o comando ls mostrar ficheiros ocultos indicamos a opcao -a.

[username@hostname ~]$ ls -a
.        Desktop    .icons    Public       .Xauthority
..       Documents  .local    Templates    .Xdefaults
.cache   Downloads  Music     .thumbnails  .xsession-errors
.config  .esd_auth  Pictures  Videos

listar permissoes dos ficheiros opcao -l. a coluna da esquerda mostra o tipo de ficheiro e os atributos de permissoes.

[username@hostname ~]$ ls -l
total 0
drwxr-xr-x 2 testeboy novonome 6 jun 28 23:13 Desktop
drwxr-xr-x 2 testeboy novonome 6 jun 28 23:13 Documents
drwxr-xr-x 2 testeboy novonome 6 jun 28 23:13 Downloads
drwxr-xr-x 2 testeboy novonome 6 jun 28 23:13 Music
drwxr-xr-x 2 testeboy novonome 6 jun 28 23:13 Pictures
drwxr-xr-x 2 testeboy novonome 6 jun 28 23:13 Public
drwxr-xr-x 2 testeboy novonome 6 jun 28 23:13 Templates
drwxr-xr-x 2 testeboy novonome 6 jun 28 23:13 Videos

comando cd

Para entrar numa pasta necessita de ter pelo menos a permissão de execução.

cd [destino]

comando Descricao
cd e cd ~ seguir para a variavel $HOME
cd / seguir para a raiz
cd - voltar a ultima pasta visitada
cd .. subir de nivel

navegar por pastas com o comando cd indicando o caminho a seguir

[username@hostname ~]$ cd / # seguir para a raiz

navegar por pastas indicado o caminho por absoluto

[username@hostname /]$ cd /usr # seguir para a pasta /usr
[username@hostname /usr]$ cd /usr/bin # seguir num caminho absoluto
[username@hostname /usr/bin]$ pwd # mostra o caminho absoluto
/usr/bin # pwd mostra isto

dois pontos representa a pasta parente. devem existir espaço em branco entre o comando cd.

[username@hostname /usr]$ cd .. # sair de /usr e seguir para a raiz /

a prompt aponta para a raiz do sistema o /.

[username@hostname /]$

manipular ficheiros ou pastas

Para criar ou apagar ficheiros é necessita de ter pelo menos a permissão de escrita.

manipular ficheiros descricao
cp copiar ficheiros e pastas
mv mover ou renomear ficheiros e pastas
rm eleminar ficheiros e pastas

copiar ficheiros - cp

copiar permissoes identicas

cp [opções] [fonte] [destino]

opcao Descricao
-i Copiar sem reescrever o ficheiro no destino
-n Não permitir que o ficheiro no destino seja reescrito
-r copiar recursivamente, util para pastas
-u copiar apenas se o ficheiro de origem for mais recente que o de destino
-attributes-only -preserve copiar atributos entre ficheiros (incluindo informações sobre o dono do ficheiro e timestamp)

renomear ou mover ficheiros - mv

Mover ficheiros significa que os ficheiros sao literalmente movidos de um local para outro. O comando mv e usado para mover ficheiros pelo sistema. Como tambem para renomear o nome dos ficheiros.

mv -i ficheiro1 ficheiro2

renomear ficheiros os dois argumentos tem de ser do mesmo tipo, pasta com pasta, ficheiro regular com ficheiro regular.

sintaxe: mv [antigoNome] [novoNome]

opcao | descricao -f | forçar -i | interativo -n | nao rescrever em ficheiro existente -u | mover apenas se source for mais novo ou quando destino esta em falta -v | modo verbose

manipular pastas descricao
mkdir criar pastas
rmdir eleminar pastas

criar pastas - mkdir

Uma pasta é usada para armazenar ficheiros de um determinado tipo. podendo ser possivel organizar por categorias e nomes

mkdir [opções] [nome] [nome2]

opcao descricao
--verbose Mostra uma mensagem para cada diretório criado. As mensagens de erro serão mostradas mesmo que esta opção não seja usada.
-p cria pastas caso nao existam

Para criar um novo diretório, você deve ter permissão de gravação.

mkdir -p ~./categoria/cenas

remover pastas - rmdir

Este comando faz exatamente o contrário do mkdir. O diretório a ser removido deve estar vazio e você deve ter permissão de gravação para remove-lo.

rmdir [nome] [nome2]

opcao descricao
-r recursivamente
-f forcar
-i confirmar iterativamente

rmdir -rf ~./categoria/cenas

criar ficheiros ocultos - touch

Para criar novos ficheiro escrevemos o comando touch, neste exemplo vamos criar um ficheiro oculto. O nome do ficheiro deve começar com o caracter . Por exemplo: .estouEscondido

touch .estouEscondido

para ver ficheiros escondidos utilizamos a opcao -a do comando ls

ls -a
