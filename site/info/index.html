Sobre Exame

Linux Essentials é a mais recente certificação do Linux Professional Institute (LPI), que se destina especialmente aos que querem iniciar, têm o objetivo de certificar os mais iniciados no sistema LINUX.

O certificado Linux Essentials foi construido para definir o conhecimento básico necessário para usar um computador Linux de forma produtiva, o objetivo é ajudar e certificar os conhecimentos de iniciantes em LINUX ou interessados nas tecnologias open source.

Os temas tratados no exame vão avaliar a compreensão do que é o software open-source, do que é o LINUX e como poderá iniciar corretamente a aventura neste mundo desconhecido.

Todo o conteúdo deste manual é uma simples preparação para o exame, escrita em lingua portuguesa. Oferece aos candidatos da certificação LINUX Essentials os conhecimentos necessários e com exemplos práticos. Tambem não deixa de ser uma boa referencia para o uso e compreensão do sistema operativo Linux.

aborda alguns conceitos mais importantes do Linux e ferramentas como:

  • introdução detalhada sobre o software livre
  • sistemas de código aberto e Linux
  • shell
  • lidar com ficheiros e scripts
  • estrututa do sistema de ficheiros
  • funcionamento e administração Linux
  • gerir utilizadores e seus privilegios
  • usar linux como cliente numa rede

Uma vez concluido com sucesso o exame, os alunos do Linux Essential estão bem preparados para continuar com mais certificações LPIC, bem como outras certificação tais como:

  • linux foundation
  • Red Hat
  • SuSE
