compilador

É o programa que converte o programa feito pelo programador em linguagem máquina. Após o processo de compilação o programa fica pronto para ser executado como um ficheiro binário.

Existem muitos compiladores no ambiente GNU/Linux, um dos mais usados é o gcc, o compilador para linguagem C.

O que são linguagens de programação?

Todos os programas e até o proprio sistema operativo, são escritos com uma ou mais linguagens de programação. Os programas também são ficheiros, mas o sistema operativo os trata como uma sequência de instrucoes e condicoes que devem ser executadas pelo processador e outros dispositivos.

Existem demasiadas linguagens de programação. Felizmente em sistemas Linux é possivel estimular a aprendizagem e desenvolver praticamente qualquer linguagem de programação.

Algumas linguagens de programção mais comuns:

C

Está associado ao sistema Unix, mas pode ser utilizada para escrever programas para quase todo tipo de computador ou dispositivo. As maiores vantagens da linguagem C são flexibilidade e desempenho. Tanto super processadores ou microcontroladores em aparelhos domesticos são programados na linguagem C.

c++

oferece caracteristicas a linguagem C para programção orientada a objectos

Java

A maior virtude do Java é a portabilidade, permite escrever um só mesmo código para ser usado em diferentes sistemas operativos. Diferente da linguagem C, um programa em linguagem C uma vez compilado so funciona no sistema operativo para o qual foi criado.

Perl

A linguagem Perl é uma linguagem de programação muito utilizada para manipular conteudos de texto. Com recursos a expressoes regulares e sintaxe na formatacao, e possıvel tratar textos e gerar relatorios como paginas de Internet (ficheiros HTML).

Script da Shell

O scripting nao e apenas mais uma linguagem de programacao, mas um ambiente de interacao com o computador. Pela linha de comando Shell, e possıvel gerar pequenos programas (scripts) que automatizam tarefas complicadas ou repetitivas.

Python

A linguagem Python é popular entre estudantes e profissionais que nao estao diretamente ligados a area de programaçao. Apesar de possuir recursos avancados, a linguagem Python e um bom comeco para quem deseja aprender a programar.

PHP

PHP é uma sigla que significa, PHP: Hypertext Preprocessor. É uma linguagem tipo script e open source. Especializada para o desenvolvimento web e que pode ser embutida dentro da linguagem HTML. O PHP é o mais utilizado em serviços HTTP.

A maioria das paginas da Internet nao sao ficheiros armazenados no servidor. Mas conteudos gerados dinamicamente pelo servidor a partir de diversas fontes, como base de dados e outros ficheiros tipo PHP com HTML e Javascript que geram vistas com informação.

Desta associação com um sistema Linux e os programas Apache, MySQL e PHP surgiu uma outra sigla LAMP.

LAMP: Linux, Apache, MySQL e PHP, a combinação destes programas ficou popular por manipular conteudos para a Internet.

SQL

Linguagem desenvolvida pela IBM que usa comandos simples baseados em palavras em ingles, para realizar pesquisas em base de dados.

Programas compilados, intrepertados, assembley

  • programas compilados: É a transformação de um programa em código fonte (programa escrito pelo programador) em linguagem de máquina (programa executável).

  • programas intrepretados: os programas sao convertidos em linguagem maquina quando executados, sao interpretados. sao mais lentos que os compilados

  • assembley: corelacao entrecodigo maquina e simbolos usado muito em leituras de baixo nivel. a leitura do código é complexa

Desenvolvimento do software por modelos:

  • Catedral acesso ao codigo fonte durante o desenvolmento a limitado a pequenos grupos de programadores. é mais estruturado e menos colaborativo

  • Bazaar: menos organizado, "caotico como um mercado onde todos vendem o mesmo" convida a colaboracao de outras pessoas
